# Micro Manual Layout

A structured multi‑page layout for compact, high‑clarity educational guides.

## Page Setup
- A4 + US Letter compatible
- 12‑column grid
- Margin: 32px
- Gutter: 16px
- Running headers and footers

## Structure
- Title block on page 1
- Table of contents (optional)
- Section headers for major divisions
- Step‑by‑step blocks for processes
- Info cards for key concepts
- Callout boxes for tips, warnings, and highlights
- Image frames for examples, diagrams, or wildlife photos

## Footer
- Page number
- Brand name or logo
