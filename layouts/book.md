# Book Layout

A structured layout for long‑form books, guides, and chaptered educational content.

## Page Setup
- A4 + US Letter compatible
- 12‑column grid
- Margin: 32px
- Gutter: 16px
- Running headers and footers
- Page numbers

## Structure
- Chapter title pages with large title block
- Section headers for major divisions
- Multi‑column text for readability
- Pull quotes for emphasis
- Info cards for key concepts
- Callout boxes for tips, warnings, and highlights
- Image frames for wildlife photos, diagrams, or illustrations
- Optional full‑bleed images for chapter openers

## Footer
- Page number
- Book title or chapter title
