# clarity-design-system
Design system for cheat sheets, micro manuals, and books — includes tokens, components, layout rules, and brand assets.

## Purpose
To ensure visual consistency, clarity, and modularity across all digital products, enabling rapid creation of high‑quality layouts in Claude Design and other tools.

## Structure
- design-tokens.json — colours, typography, spacing, radii, shadows
- components/ — reusable UI and layout components
- layouts/ — page templates for cheat sheets, manuals, and books
- icons/ — icon guidelines and references
- brand/ — brand notes, tone, and usage rules

## Usage
Link this repository in Claude Design under “Link code on GitHub” to allow Claude to apply the design system automatically.
