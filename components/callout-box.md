# Callout Box

Used for warnings, tips, notes, or highlights.

Variants:
- Tip (accent)
- Warning (warning)
- Danger (danger)

Structure:
- Border-left: 4px in variant colour
- Background: neutralLight
- Padding: md
- Optional icon at the start
- Body text uses standard body font

Usage:
Ideal for cheat sheets, micro manuals, and book sidebars.
